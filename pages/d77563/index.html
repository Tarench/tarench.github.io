eg2.爬取页面新闻的标题和链接

url = 'https://news.baidu.com/'
response = requests.get(url)
# 解析网页
soup = BeautifulSoup(response.text, 'html.parser')
# 定位div hotnews
divs = soup.find('div', class_='hotnews')
for n in range(0,5):
    # 定位a标签
    a = divs.find('li', class_='hdline{}'.format(n)).find('a')
    title = a.get_text()
    herf = a.get('herf')
    print(title)
    # 导出到文件中
    with open('test.txt', 'a', encoding='utf-8') as f:
        f.write("标题：" + title + "\n")
        f.write("链接：" + str(herf) + "\n")

# 超时设置

  • 利用 timeout 变量来配置最大请求时间，如果在timeout时内请求内容没有返回， 将产生一个timeout的异常
import requests
r = requests.get('http://www.baidu.com', timeout = 0.5)
print(r.status_code)

  • 如果远端服务器很慢，可以传入一个None作为timeout值，使得requests永远等待
  • timeout 仅对连接过程有效：只限制请求的时间，与响应体的下载无关。即：当返回的 response 包含很大内容，下载需要一定时间，与timeout没有关系

# 获取的内容

  • 网页文本：html文档、json、csv等
  • 图片、音频、视频：二进制文件，保存为相应的格式
r = requests.get('图片url')
print(r.content) # 二进制文件用content
# 保存图片
with open('img.jpg', 'wb') as f:
    f.write(r.content)

# 爬取网页的通用代码框架

import requests  # 加载Requests库

def get_html(url):  # 定义get_html函数
    try:
        r = requests.get(url, timeout=40)  # 设定get函数参数，超时限制40s
        r.raise_for_status()  # 有效判断网络连接状态，错误将捕获异常
        r.encoding = r.apparent_encoding  # 设置编码格式
        return r.text  # 返回网页文本内容
    except:
        return "产生异常"  # 返回异常提示

if __name__ == "_main_":  # 直接运行时（非以模块导入）
    url = "http://www.baidu.com"  # 给url赋值
    print(get_html(url))  # 打印函数内容

# 请求

  • 请求是由客户端向服务端发出的，包括：request method请求方式；request URL请求网址；request header请求头；request Body请求体
  1. request methond：主要有GET, POST, HEAD, PUT DELETE五种
  2. request URL：统一资源定位符，即网址。URL可以确定一张网络图片、一个音频、一个网页文档等，其包含的信息指出了文件的位置以及浏览器如何处理它
  3. request header：包含请求时的头部信息，例如User-Agent（指定浏览器的请求头），Cookies、Host等
  4. request body：额外携带的数据，例如登录表单提交的登录信息

  • 正常访问网站的USer-Agent：
  • 用爬虫访问网站的USer-Agent：

User-Agent：向访问网站提供你所使用的浏览器类型及版本、操作系统及版本、浏览器内核、等信息的标识

  • 有些网站禁止爬虫访问，添加浏览器的User-Agent信息
import requests
headers = {'User-Agent':"Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/106.0.0.0 Safari/537.36 Edg/106.0.1370.47"}
r = requests.get('http://baidu.com', headers = headers)
print(r.text)
